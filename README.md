# 藝吧查表演

https://winniecat777.github.io/performanceSearch/

本專案僅為學習用途，不做任何商業使用

## 專案介紹

提供台中地區近期表演活動、課程與場地資訊的查詢平台

## 開發工具

- `Vite` 建構開發環境
- `Bootstrap 5` 搭配 `Sass` 進行網頁切版
